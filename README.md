# Sudoku Hub

A platform for Sudoku enthusiasts to practice, learn and enjoy!

This application is built using Ruby on Rails 6 (6.0.2)

## Setup

- `rvm install 2.6.5`
- `rvm use 2.6.5`
- `rvm gemset create sudoku`
- `rvm gemset use sudoku`
- `bundle install`
- `rails db:setup`
- `rails db:migrate`
